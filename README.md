# bonay-santiago.com

Personal portfolio and CV website for Santiago Bonay — Senior AI Software Engineer.

## Stack

- **Next.js 14** (App Router) + TypeScript
- **Tailwind CSS** — utility-first styling, editorial Japanese-inspired design
- **next-intl** — i18n with 3 locales (EN / ES / JA)
- **Framer Motion** — page transitions and scroll-triggered animations
- **Vercel** — deployment

## Pages

| Route | Content |
|-------|---------|
| `/` | Hero + About Me + skills grid |
| `/experience` | Professional timeline, education, certifications |
| `/projects` | Featured project cards (CPM Agent Accelerator) |
| `/contact` | Email, LinkedIn, GitHub links |

## Development

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
npm run lint    # ESLint
```
